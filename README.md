## 📌 Project Description
This project is a **Pro Stopwatch Web Application** built as Task 02 for SkillCraft Technology Internship. 
It includes start, pause, reset, and lap functionality with split time and total time tracking for each lap.
## 🎯 Objective
To create a fully functional, responsive stopwatch that allows users to accurately measure time, record laps, and export data.
## ✨ Features Implemented
- **Start / Pause / Reset Controls**: Core stopwatch functionality
- **Lap Tracking**: Record split time and total time for each lap
- **Fastest/Slowest Lap Highlight**: Automatically highlights best and worst laps
- **Live Current Lap Timer**: Shows running time of current lap
- **Export to CSV**: Download all lap records as a CSV file
- **Keyboard Shortcuts**: `Space` = Start/Pause | `L` = Lap | `R` = Reset
- **Dark/Light Theme Toggle**: Switch between themes for better UX
- **Responsive Design**: Works seamlessly on desktop and mobile
- **Accurate Timing**: Updates every 10ms for precision
## 🛠️ Technologies Used
- **HTML5** - Structure and layout
- **CSS3** - Styling, Flexbox, Grid, Glassmorphism, Animations
- **JavaScript (ES6)** - Timer logic, DOM manipulation, CSV export, Local state
